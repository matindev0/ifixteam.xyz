# iFix Tech Website

## Overview
A static portfolio website for iFix Tech, showcasing web development and Discord bot services.

## Project Structure
- `index.html` - Main landing page
- `discord.html` - Discord server information page
- `register.html` - Registration page
- `css/` - Stylesheets (style.css, aos.css, contact.css, register.css)
- `js/` - JavaScript files (aos.js, contact.js, register.js)
- `images/` - Image assets
- `img/` - Additional image assets

## Running the Project
This is a static website served via Python's HTTP server on port 5000.

## Deployment
Configured for static deployment, serving files from the root directory.
