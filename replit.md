# iFix Tech Website

## Overview
A static portfolio website for iFix Tech, showcasing web development and Discord bot services with an electronic products catalog.

## Project Structure
- `index.html` - Main landing page with hero section, skills, and services
- `products.html` - Electronic products catalog with 8 items and search functionality
- `discord.html` - Discord server information page
- `register.html` - Registration page
- `css/` - Stylesheets
  - `style.css` - Main styles for the site
  - `products.css` - Styles for products page
  - `aos.css` - Scroll animation styles
  - `contact.css` - Contact form styles
  - `register.css` - Registration form styles
- `js/` - JavaScript files (aos.js, contact.js, register.js)
- `images/` - Image assets
- `img/` - Additional image assets

## Features
- Responsive design with dark theme and blue accents
- Products page with 8 electronic items (dual pricing: regular and final price)
- Real-time search functionality for products
- Scroll animations using AOS library
- Contact form integration via Formspree

## Running the Project
This is a static website served via Python's HTTP server on port 5000 (0.0.0.0:5000).

## Deployment
Configured for static deployment, serving files from the root directory.
